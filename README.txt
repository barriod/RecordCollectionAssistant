1. To run RecordCollectionAssitant locally:
   Open project solution in Visual Studio. (C#)
   Run.

2. List of files to run system:
   Within src/RecordCollectionAssistant/RecordCollectionAssistant/ is a folder
   for accounts and their saved information with text files. These are within the 
   AccountsDatabase folder. Ensure any account information that user wants is 
   saved properly there.

   Additionally, there are a list of Images and Barcodes to test the album
   art uploading and add record by barcode functinality of the system. Use
   the images in the folder and the Barcodes.txt file to test barcode functionality.
